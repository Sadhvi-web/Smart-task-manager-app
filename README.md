# Smart Task Manager App

The Smart Task Manager is a frontend web application designed to help users manage daily tasks through a clean and responsive user interface. The application focuses on core task operations and frontend best practices.

## Features
- Create, edit, and delete tasks
- Responsive UI for desktop and mobile devices
- Task persistence using browser LocalStorage
- Simple and user-friendly interface

## Tech Stack
- HTML
- CSS
- JavaScript
- Vite
- pnpm
- Bootstrap

## Project Structure
```text
├── index.html            # Main HTML file
├── style.css             # Application styles
├── script.js             # JavaScript logic
├── package.json          # Project configuration and scripts
├── pnpm-lock.yaml        # Dependency lock file
├── template_config.json  # Template configuration
└── readme.md               # Development notes and tasks
```
## File Description Inventory
index.html: The main structure of the web application, including the layout and components.
package.json: Manages project dependencies and scripts.
script.js: Contains the JavaScript logic for task management and UI interactions.
style.css: Styles the application, ensuring a modern and responsive design.
template_config.json: Configuration details for any templates used in the project.
todo.md: Documentation outlining development tasks and design guidelines.
## Technology Stack
Frontend: HTML, CSS, JavaScript
Frameworks: Bootstrap for responsive design
File Management: EJS for templating (future use)
Storage: LocalStorage for data persistence
## Usage
Install Dependencies: Use pnpm install to install required packages.
Build: No build step is required for this static application.
Run: Open index.html in a web browser to view the application.
