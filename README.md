# Smart-task-manager-app
The Smart Task Manager is a frontend web application designed to streamline task management through a responsive user interface. It offers dynamic create, read, update, and delete (CRUD) functionalities, ensuring a consistent experience across devices. Built using HTML, CSS, JavaScript, and Bootstrap, it integrates with REST APIs for backend functionality.

Project Module Description
The application consists of several key modules:

Task Management: Allows users to create, edit, and delete tasks.
User Interface: Responsive design that adapts to desktop, tablet, and mobile views.
Statistics Dashboard: Displays real-time statistics on tasks (total, pending, completed).
LocalStorage: Persists task data across sessions.
REST API Integration: Prepares for future backend connectivity.

Technology Stack

Usage
Install Dependencies: Use pnpm install to install required packages.
Build: No build step is required for this static application.
Run: Open index.html in a web browser to view the application.
